# Logos des références clients

Déposer ici les logos officiels des références (promoteurs + institutions),
puis renseigner le chemin dans `src/lib/data/groupe.ts` → `clientReferences[].clients[].logo`.

Format recommandé : SVG ou PNG transparent, ~320×128 px, fond transparent.

Exemple :
  { name: 'Grande Mosquée d'Alger', logo: '/images/references/grande-mosquee-alger.svg' }

Noms de fichiers suggérés :
  grande-mosquee-alger, ministere-habitat, ministere-defense, ministere-justice,
  ministere-interieur, ministere-tourisme, benamar-promotion, chili-promotion,
  provico-promotion, hntp, spil
