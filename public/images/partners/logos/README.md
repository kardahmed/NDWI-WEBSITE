# Logos des fournisseurs & partenaires

Déposer ici les logos, puis renseigner le chemin dans `src/lib/data/partners.ts` :
- partenaires vedettes → `partners[].logo`
- roster complet → `partnerRoster[].items[].logo`

Format recommandé : SVG ou PNG transparent, ~240×96 px, fond transparent.

Roster à couvrir :
  Machines : omag, biesse, scm, barbieron, essipigi
  Panneaux/matériaux : aran-cucine, ica-group, fantoni, finsa, kastamonu, egger, alfa-wood, lamitex
  Quincaillerie : blum, hettich, dtc, agb, hope
